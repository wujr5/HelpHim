# HelpHim
HelpHim网页应用的github仓库，基于meteor。

运行和部署：

1. git clone https://github.com/wujr5/HelpHim.git

2. cd HelpHim

3. git checkout work-tree

4. meteor

贡献到work-tree

1. git add -A

2. git commit -m 'your comment'

3. git push origin work-tree


分工：

家荣：one_task

庆锋：function_area

霭霖：new_list

秋霞：publish_task

文杰：home
